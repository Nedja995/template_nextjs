# Template Web App

- NextJS, Typescript, Redux, StyledComponents, Jest

## Using

1. Install the dependencies

```
npm install
```

2. Run dev

```
npm run dev
```

#### Run tests

```
npm run test
```

#### Tips
- Localized strings in `src/config/i18n.ts'
